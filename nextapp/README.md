# Next.js Application Workspace

This directory contains the main Next.js 15 application codebase for the full-stack developer portfolio platform.

## 📖 Main Documentation

For full setup instructions, local development scripts, workflows, system architecture, database configuration, and licensing terms, please refer to the **[Root README.md](../README.md)**.

## 🛠️ Quick Commands

Although you can run commands directly from the root workspace using the root `package.json` scripts, you can also run them here:

- **Install dependencies:** `npm install`
- **Start development server:** `npm run dev`
- **Build production bundle:** `npm run build`
- **Seed database:** `npx prisma db seed`
- **Sync database schema:** `npx prisma db push`
